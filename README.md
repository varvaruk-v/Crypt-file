# Cryptor

Cryptor is a Python tool that can encrypt(decrypt) text/file or hide text in PNG.

## Installation

Use the package manager [pip](https://pypi.org/project/Cryptortool/) to install Cryptor.

```bash
pip install Cryptortool
```

## Usage
### 1) Run
```bash
cryptor
```
